# Space Engineers Scripting Reference

A living reference for writing C# scripts for the Space Engineers Programmable Block.

## Contents

- `docs/scripting-reference.md` - Comprehensive scripting guide and API reference
- `scripts/` - Our custom scripts (to be added as we build things)

## Quick Start

1. Place a **Programmable Block** on your grid
2. Enable **Experimental Mode** in game settings
3. Enable **In-game Scripts** in world settings
4. Open the Programmable Block, paste your script, click "Check Code" then "Run"
